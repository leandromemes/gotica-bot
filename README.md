## 📖 Sobre o Projeto

A **Gótica Bot** é uma evolução focada em comandos de entretenimento e utilitários. Priorizamos a interação entre membros com sistemas de **Relacionamentos** (casar, namorar, trair) e **Economia Real** (ganhar dinheiro, apostar e trabalhar), tudo configurado para ser intuitivo e divertido. Desenvolvido e mantido por Leandro, utilizando a base Takeshi.
